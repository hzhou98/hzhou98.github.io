Honglei Zhou

I'm a research assistant professor and senior software engineer at Jiangsu Province Hospital in Nanjing, where my primary focus is on developing bioinformatics analysis platforms, conducting research data analysis, and collaborating with physicians to address real-world challenges.

Before I join Jiangsu Province Hospital, I worked as senior software engineer in MarkLogic in San Mateo, where I mainly involved in developing an AWS Spot Fleet-based MarkLogic DB SaaS service.

I also have sereral years of experience in Start-up companies. From 2022 to 2024, I worked as part-time software engineer in Paigo to design and develop the Paigo backend customer service API, and implement frontend dashboard feature. From 2018 to 2020, I joined Bliss Motors as a founding member, working on the full development stack of Bliss Motors' services and creating a chatbot named Telle AI.

I earned my Master’s degree in Computer Science from the University of Chicago in 2018. Work Experience

I have worked on multiple projects in both academia and industry, as listed below.

Bioinformatics Analysis Platform (Jiangsu Province Hospital,2021-2024)
Tech stack: R, Python, Flask, JavaScript, Vue.js, OpenStack, Docker, cAdvisor, Graphite

We developed a bioinformatics analysis platform with an intuitive dashboard, allowing scientists to conduct data analysis without needing to write code.

Paigo SaaS Billing (Paigo,2022-2024)
Tech stack: Kubernetes,Typescript,NestJS,Redis,BullMQ

Paigo features an agent that collects resource metrics from clients, and it also supports clients sending resource metrics via REST APIs. The platform provides a dashboard where clients can define resource units used for billing their customers.

Spot Fleet Option for provisioning MarkLogic Cluster on AWS (MarkLogic,2020-2021)
Tech stack: Python, AWS SDK, Atlassian

To reduce costs while still meeting client service-level agreements, I explored using AWS Spot Fleets as a more cost-effective alternative for clusters.

Bliss Motors Chatbot SaaS (Blis Motors,2018-2020)
Tech stack: JavaScript, Vue.js, HTML/CSS, Python, scrapy, scrapy-redis, PostgreSQL, MongoDB, Redis, AWS, Django, Django Rest-Framework, OAuth, Twilio, Ejabbered

Our chatbot has a frontend built using JavaScript with the Vue.js framework, allowing it to be embedded in the website as an iframe. The backend consists of multiple microservices that handle chat functionality and other services.

Job Level Monitoring System (The University of Chicago,2018)
Tech stack: Python, PostgresSQL, cAdvisor, STATSD, Graphite

We implemented a job-level monitoring system within the platform in HPC cluster. This system monitors key metrics—such as CPU usage, memory, I/O, and network activity—at the job level, as each job runs within a container.
